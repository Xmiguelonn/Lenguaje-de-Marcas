Acerca del Proyecto

Descripción General del Proyecto

Este proyecto es una aplicación web sencilla desarrollada como parte del curso de 1º DAW en el módulo de "Lenguaje de marcas". Su objetivo principal es practicar el uso de información dinámica a través de archivos JSON y llamadas a APIs

Objetivos del Desarrollo

  • Aprender y aplicar los fundamentos del desarrollo web Front-end (HTML, CSS, JavaScript).
  • Familiarizarse con el uso de librerías como Bootstrap para un diseño responsivo.
  • Uso de datos externos con llamadas a APIs públicas
  • Crear una interfaz de usuario intuitiva y fácil de usar.

APIs Utilizadas

Por falta de tiempo no se han podido usar APIs por lo que hemos tenido que trabajar con ficheros JSON estáticos

Tecnologías Empleadas

  • HTML: Para la estructura y contenido de la página web.
  • CSS: Para el diseño y la presentación visual de la aplicación.
  • JavaScript: Para la interactividad con el usuario y ayuda con el desarrollo y el uso de la información externa.
  • Bootstrap 5: Para facilitar el diseño responsivo y el estilo de los elementos.
  • JSONGuardar información externa.
